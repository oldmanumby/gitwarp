# gitswapForged

A clean, client-side web application that acts as an interactive version of the GitHub URL-Swap Cheat Sheet. Users can paste a GitHub repository URL, and the app will dynamically generate a grid of cards providing "swapped" URLs that unlock different superpowers for that repository.

Based on Hyperautomation Labs Cheat Sheet.
Find more at: [https://hyperautomationlabs.co](https://hyperautomationlabs.co)

## Features

- **Instant Swaps:** Generate URLs for services like gitingest, github.dev, and bolt.new with one click.
- **Copy to Clipboard:** Quickly copy the generated URLs to your clipboard.
- **Responsive Design:** A beautiful, glassmorphism-inspired UI that works perfectly on desktop and mobile.

## Setup & Installation

This project is built using vanilla JavaScript and Vite.

**1. Clone the repository and navigate into it:**

```bash
cd APP-gitswapForged
```

**2. Install dependencies:**

```bash
npm install
```

**3. Run the development server:**

```bash
npm run dev
```

**4. Build for production:**

```bash
npm run build
```
